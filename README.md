# NASA EONET Disaster Intelligence Dashboard

## Overview
An interactive Power BI dashboard that analyzes global disaster events using NASA EONET data. The project helps monitor incidents, assess risk levels, identify disaster trends, and support data-driven decision-making through visual analytics.

## Tools Used
- Power BI
- Power Query
- DAX
- SQL
- Microsoft Excel
- NASA EONET API

## Project Files
- Disaster_management.pbix
- disaster_data.xlsx
- disaster_SQL.sql
- NASA-EONET-Disaster-Intelligence-Dashboard.pptx

## Key Features
- Global Disaster Monitoring
- Risk Assessment Analysis
- Geographic Incident Mapping
- KPI Tracking
- Category-wise Disaster Analysis
- Interactive Filters and Drilldowns

## Skills Demonstrated
- Data Cleaning
- Data Transformation
- SQL Querying
- Data Visualization
- Dashboard Development
- Business Intelligence Reporting
